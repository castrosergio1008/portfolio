# 💼 Portfolio - Sergio Castro

My personal portfolio website showcasing my projects and skills as a Full-Stack Developer.

## ✨ Features

- 🎨 **Modern Dark Theme** - Sleek, professional design
- 📱 **Fully Responsive** - Works on all devices
- ⚡ **Fast Performance** - Built with Vite
- 🎯 **Smooth Animations** - CSS animations and transitions
- 🔗 **Project Showcase** - Links to live projects and GitHub repos
- 📊 **Skills Visualization** - Interactive skill bars

## 🛠️ Tech Stack

- **Framework:** React 19
- **Styling:** Tailwind CSS 4
- **Build Tool:** Vite 8
- **Icons:** Lucide React

## 🚀 Getting Started

```bash
# Clone the repository
git clone git@github.com:castrosergio1008/portfolio.git

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`

## 🎨 Customization

### Colors
Edit `src/index.css` to change the gradient colors:
```css
.gradient-text {
  background: linear-gradient(135deg, #3b82f6 0%, #8b5cf6 100%);
}
```

### Content
Update the content in each component file under `src/components/`.
